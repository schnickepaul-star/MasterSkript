==================================================
MasterSkript v38.0 - Documentation
==================================================

MasterSkript is a powerful yet intuitive UI and logic prototyping language.

---
ELEMENTS (Usage: show(...))
- show(text("...")): Displays text on the screen.
- show(button("...")): Displays a button with the specified label.
- show(checkbox): Displays a checkbox.
- show(radio): Displays a radio button.
- show(image("path")): Displays an image from the provided path.
- show(var("name")): Displays the value of a variable.
- show(windowcontrols): Adds window control buttons (Close, Minimize, Maximize).

---
LOGIC
- If button("Label") clicked: Trigger actions based on button interaction.
- Set target("name") to ("value"): Updates the content of an element.
- print.console("..."): Outputs text to the integrated console.
- calculate(expression): Performs math and prints the result to the console.
- True / False: Boolean logic support.
- Else: Fallback logic for conditions.

---
VARIABLES
- let name = "value": Assigns a value to a variable (e.g., let greeting = "Hello").

---
STRUCTURAL & IMPORTS
- window: Initiates the scope. All UI elements must be inside this block.
- .class=name: Adds a specific class to an element for styling.
- Imports (Required): import masterskript as app, import masterUi as style.
- Imports (Optional): import python as py, import cpp, import tkinter as style.

---
STYLELANGUAGE (StylerSkript)
To customize elements, use the style block. Ends with !style.
- style for element("name"):
    - color="hex/name";
    - position = x(val), y(val); (in px)
    - border = val px;
    - Remove.UI: Removes default styling.
    - margin(all/top/bottom/left/right) = val px;
- hover: !hover block for interaction effects.
- animation: Custom movement, rotation, or scaling (e.g., go 20px for x in 2sec).
- Rotate(x/y) = degrees;
- Scale(x/y) = pixels;
- shadow = color:hex position x(val) y(val);

---
TKINTER INTEGRATION
- To use native Tkinter styling, use:
  style tkinter for element("name"):
  ...
  !style

---
MULTILINGUAL SCRIPTS
- script [lang] for element("name"): Allows embedding logic (e.g., Python, C++)
  directly to an element. Ends with !script.
